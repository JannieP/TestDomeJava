If you are having issues while using TestDome try these immediate recovery steps:

  1. Make sure you are connected to the Internet:
  2. Try using private/incognito mode in your browser (or starting the browser with all extensions and add-ons disabled). To open a new private/incognito window:
    • In Chrome and Edge: Press Ctrl+Shift+N on Windows/Linux, or ⌘+Shift+N on macOS
    • In Firefox: Press Ctrl+Shift+P on Windows/Linux, or ⌘+Shift+P on macOS
  3. Update your browser to the latest version and then reopen the test in private/incognito mode
    • We recommend using Chrome, Edge, or Firefox.
    • Here’s how to update Chrome, Edge, and Firefox.
  4. Ask the test owner that invited you to the test to re-invite you using this same email address (at no cost for them) and make sure you try the sample questions before you start the test to see if everything is working.

It’s safe to close this tab and log in to the test using the same link you used before to start it.
If you were invited via URL without email verification, then the test will start from the beginning.

If you are still having issues with your test, please contact us at support@testdome.com and we’ll answer you as soon as possible (usually within 1 business day).

We apologize for the inconvenience.
